Requirement
===========

:Python: 3.7.1
:Django: 2.1.7


Quick start
===========
1. プロジェクトのクローン::

    git clone https://github.com/pistgroup/notech.git

2. モデルのmigrate::

    python manage.py migrate

3. うごかす::

    python manage.py runserver
